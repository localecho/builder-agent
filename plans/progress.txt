- `node src/index.js status` - Show repo status
- `node src/index.js update-deps` - Scan and update dependencies
- `node src/index.js builds` - Show build status
- `node src/index.js changelog` - Generate changelog
- `node src/index.js release` - Create release
- `node src/daemon.js` - Start continuous monitoring

## Environment Variables
- GITHUB_TOKEN - Required for API access
- TARGET_REPOS - Comma-separated owner/repo list
- SLACK_WEBHOOK - Optional Slack notifications
- DISCORD_WEBHOOK - Optional Discord notifications
- DEPLOY_WEBHOOK - Deployment trigger URL
- DEPLOY_COMMAND - Deployment shell command
